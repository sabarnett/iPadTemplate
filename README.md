# iPadTemplate

This is a basic template for iPad master/detail apps. In order to use the template ou should:

* Clone this repo to your local machine.
* Rename the cloned project.
* Review the instructions for the customisation you need to do.

The project runs and provides:

* A launch screen that adapts to portrait and landscape.
* A home screen that uses `NavigationSplitView` to control the app.
* A sidebar with a dummy list.
* A detail view that reacts when a sidebar item is selected.
* A popup `about` view with basic app related information.
* A template `settings` view that supports changing the app color scheme between light/dark/system mode.
* An IconDesigner icon file.

## Renaming the project

You could rename the cloned project manually, but that would probably take longer than it doesto create the templat in the first place. There are, however, nice tools that will automate the process for you. I would suggest you look at Stewart Lynch's xcrename tool because it is simple to use and easy to install. Take a look at it [here](https://github.com/StewartLynch/XcodeProjectRenamer).
